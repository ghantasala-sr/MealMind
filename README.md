## üöÄ Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/ghantasala-sr/MealMind.git
   cd MealMind/meal_mind_streamlit
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the application:
   ```bash
   streamlit run Home.py
   ```

## üç¥ Forking This Repository

This repository is designed to be fork-friendly! To sync your fork:

### Quick Sync (GitHub UI)
1. Go to your fork on GitHub
2. Click the "Sync fork" button
3. Click "Update branch"

### Manual Sync (Git CLI)
```bash
git remote add upstream https://github.com/ghantasala-sr/MealMind.git
git fetch upstream
git merge upstream/main
git push origin main
```

**Having trouble syncing?** See our [Fork Sync FAQ](FORK_SYNC_FAQ.md) for solutions to common issues.
